Échecs collaboratif

Votez pour un coup avec votre équipe

Comment ça marche ?

Deux équipes s’affrontent aux échecs : les Blancs et les Noirs. Plusieurs personnes peuvent être dans chaque équipe. À chaque tour, toute l’équipe au trait vote pour le coup à jouer (bouton ou glissement sur l’échiquier). Les votes sont visibles par tout le monde. Quand le chronomètre de 30 secondes se termine, le coup le plus voté est joué automatiquement. Tu peux utiliser une fois par partie un double vote. Utilise le chat pour te coordonner avec ton équipe.
